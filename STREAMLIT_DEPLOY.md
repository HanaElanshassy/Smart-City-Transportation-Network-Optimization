# Streamlit Community Cloud Deployment

This project is ready to deploy for free on Streamlit Community Cloud.

Use these settings:

```text
Repository: HanaElanshassy/Smart-City-Transportation-Network-Optimization
Branch: main
Main file path: app.py
Python version: 3.11
```

Deploy from:

```text
https://share.streamlit.io/
```

Streamlit Cloud will install Python dependencies from `requirements.txt` and
system packages from `packages.txt`.
